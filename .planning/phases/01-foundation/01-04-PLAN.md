    - main(dry_run=False, skip_midi=False, soundfont=Path("tools/TimGM6mb.sf2")): iterates raw_assets/icons/*.ico and raw_assets/sounds/*.mid; with dry_run=True, only prints file list and exits 0 without writing

  <action>
    Create convert_assets.py in the project root with these components:

    Imports: argparse, pathlib, shutil, subprocess, sys, and from PIL import Image.

    Helper function convert_ico(src: pathlib.Path, dst: pathlib.Path, size: int = 128) -> bool:
      Wrap the Pillow open/convert/resize/save pipeline in a try/except that catches PIL.UnidentifiedImageError and OSError. On error: print a clear message including the source filename and the exception text; return False. On success: return True.
      Use Image.NEAREST for resize (not LANCZOS, not BICUBIC — pixel art must not be blurred).
      Call dst.parent.mkdir(parents=True, exist_ok=True) before saving.

    Helper function convert_midi(midi_path: pathlib.Path, wav_path: pathlib.Path, soundfont_path: pathlib.Path) -> bool:
      First: if shutil.which("fluidsynth") is None: print a two-line error message identifying FluidSynth as not installed, then print "Install: winget install FluidSynth.FluidSynth  OR  conda install -c conda-forge fluidsynth"; call sys.exit(1).
      Then: call subprocess.run(["fluidsynth", "-ni", str(soundfont_path), str(midi_path), "-F", str(wav_path), "-r", "44100"], check=True). Wrap in try/except subprocess.CalledProcessError; on error print a message and return False. Return True on success.

    Helper function copy_wav(src: pathlib.Path, dst: pathlib.Path) -> bool:
      dst.parent.mkdir(parents=True, exist_ok=True). shutil.copy2(str(src), str(dst)). Return True.

    Main function def main():
      Parse args with argparse: --soundfont (pathlib.Path, default Path("tools/TimGM6mb.sf2")), --skip-midi (store_true), --dry-run (store_true).
      Validate soundfont path: if not args.dry_run and not args.skip_midi and not args.soundfont.exists(): print warning that soundfont not found; sys.exit(1).
      Define: RAW_ICONS = pathlib.Path("raw_assets/icons"), RAW_SOUNDS = pathlib.Path("raw_assets/sounds"), OUT_SPRITES = pathlib.Path("assets/sprites"), OUT_SOUNDS = pathlib.Path("assets/sounds").
      If not dry_run: create OUT_SPRITES and OUT_SOUNDS via mkdir(parents=True, exist_ok=True).
      ICO loop: for ico in sorted(RAW_ICONS.glob("*.ico")) + sorted(RAW_ICONS.glob("*.ICO")): dst = OUT_SPRITES / (ico.stem.lower() + ".png"). If dry_run: print(f"[DRY RUN] {ico} → {dst}"); else: call convert_ico(ico, dst).
      WAV copy loop: for wav in sorted(RAW_SOUNDS.glob("*.wav")) + sorted(RAW_SOUNDS.glob("*.WAV")): dst = OUT_SOUNDS / (wav.stem.lower() + ".wav"). If dry_run: print(f"[DRY RUN] {wav} → {dst}"); else: call copy_wav(wav, dst).
      MIDI loop: if skip_midi: print("SKIPPING MIDI: --skip-midi flag set"); else: for mid in sorted(RAW_SOUNDS.glob("*.mid")): dst = OUT_SOUNDS / (mid.stem.lower() + ".wav"). If dry_run: print(f"[DRY RUN] {mid} → {dst}"); else: call convert_midi(mid, dst, args.soundfont).
      Print summary line at end: "Done. Converted N ICO files, M WAV files, K MIDI files." (or equivalent for dry-run mode).

    Bottom of file: if __name__ == "__main__": main()

    IMPORTANT: The output filename for each ICO is ico.stem.lower() + ".png". This establishes the key naming convention that assets.py uses to load surfaces (the key is the stem lowercased). Example: GEvil.ico → gevil.png → loaded in assets.py with key "gevil".

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| --soundfont CLI arg → filesystem path | User-supplied path reaches FluidSynth subprocess; must be validated |
| raw_assets/icons/ → Pillow → assets/sprites/ | ICO files are processed by Pillow; malformed files should not crash the script |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-07 | Tampering | --soundfont path traversal | mitigate | Validate soundfont path with pathlib.Path(args.soundfont).resolve(); check .exists() before passing to FluidSynth subprocess. No shell=True in subprocess call (uses list args). |
| T-01-08 | Spoofing | Malformed ICO file | mitigate | Pillow raises PIL.UnidentifiedImageError; convert_ico() wraps in try/except, prints clear error with filename, continues to next file. Script exits with non-zero code if >0 ICO files failed. |
| T-01-09 | Denial of Service | Extremely large ICO file | accept | ICO files are from known local VB6 repo; 32x32 ICOs are ~1KB each; no network download; no size risk |
| T-01-SC | Tampering | subprocess FluidSynth | mitigate | subprocess called with list args (not shell=True); soundfont path validated before call; no user-controlled string concatenation in command |
</threat_model>
